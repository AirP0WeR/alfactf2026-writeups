# Долбит нормально

- **Категория:** Forensics
- **URL/Artifacts:** https://alfactf.ru/tasks/dolby
  - `resona_subwoofer_manual.pptx` (1.8MB PowerPoint presentation)
- **Flag:** `alfa{pls_restore_deleted_info_from_pptx}`

## Условие

Новую рампу поставили только вчера: свежий фанерный лист, идеальный радиус, запах дерева — всё говорит о том, что сейчас будет лучший проезд недели. Но стоит разогнаться, и покрытие гуляет, как настил на сыром бревне: в одном месте прогибается, в другом рябит, будто стиральная доска.

Оказывается, ночью кто-то установил под рампой сабвуферы под опен-эйр и забыл их выключить. Долбит нормально, но поверхность идёт волнами. Сделайте потише, чтобы наконец кататься, а не сёрфить.

Инструкция к сабвуферам: resona_subwoofer_manual.pptx

## Наблюдения

- Файл PowerPoint со 100 слайдами, содержащий инструкцию к сабвуферу
- Подсказка: "Сделайте потише" предполагает уменьшение "громких" элементов
- Презентации PowerPoint являются ZIP-архивами, содержащими XML-файлы
- Скрытая информация может храниться в невидимых текстовых элементах

## Решение

### Первоначальный анализ

Задача предоставляет большую презентацию PowerPoint (`resona_subwoofer_manual.pptx`) и намекает на необходимость сделать что-то "потише". Это предполагает задачу стеганографии, где информация спрятана внутри презентации.

### Исследование структуры PowerPoint

Файлы PowerPoint (.pptx) являются ZIP-архивами, содержащими XML-файлы. Каждый слайд хранится как `ppt/slides/slideN.xml`. При 100 слайдах для проверки нам нужно искать аномалии.

### Обнаружение слайда 73

Слайд 73 озаглавлен "ВЫКЛЮЧАЕМ — Экстренное отключение питания". Этот слайд содержит скрытый флаг в хитрой двухслойной стеганографии:

**Слой 1: Видимый "громкий" текст**
- 6 больших текстовых блоков (размер 6000-6600, тёмные цвета)
- Расположены для написания `a l f a { }` при чтении слева направо
- Они представляют "громкий" звук, который нужно сделать тише

**Слой 2: Скрытый "тихий" текст**
- 7 текстовых блоков, расположенных внутри скобок `{ }`
- Используют цвет `#F4F2EE`, который точно совпадает с фоном слайда
- Меньший размер (3200), делающий их невидимыми невооружённым глазом
- При сортировке по X-позиции, составляют: `pls_restore_deleted_info_from_pptx`

### Извлечение флага

Решение парсит slide73.xml для:

1. Извлечения всех текстовых форм с их позициями и цветами
2. Фильтрации форм, расположенных в области "флагового холста" (Y координаты 6.7-7.2M EMU)
3. Разделения видимого текста (тёмные цвета) от скрытого текста (цвет фона)
4. Сортировки обеих групп по X-позиции для получения правильного порядка чтения
5. Объединения: `alfa{` + скрытый текст + `}` = финальный флаг

### Техническая реализация

```python
# Парсим slide73.xml из ZIP PPTX
with zipfile.ZipFile(PPTX) as z:
    slide73 = z.read("ppt/slides/slide73.xml").decode()

# Извлекаем формы с позициями и цветами
shapes = re.findall(r"<p:sp>.*?</p:sp>", slide73, flags=re.DOTALL)

# Разделяем видимый от скрытого текста на основе совпадения цвета с фоном
if BG_COLOR in colors:  # F4F2EE = фон слайда
    hidden.append((x, text))
else:
    visible.append((x, text))

# Восстанавливаем флаг: видимая рамка + скрытое содержимое
flag = frame.replace("{", "{" + body, 1)
```

Флаг объединяет "громкую" видимую структуру со "скрытым" тихим сообщением, что идеально соответствует аудио-метафоре задачи — уменьшить шум, чтобы выявить сигнал.
